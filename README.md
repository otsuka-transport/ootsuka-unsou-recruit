# 次世代の運送会社で、未来を走る。｜大塚運送 採用LP

このフォルダは、大塚運送の若手ドライバー採用LPを管理するためのHTML/CSS/JS一式です。
老舗の安心感と次世代の挑戦感を伝え、LINE相談につなげる採用LPとして作成しています。

## ファイル構成

- `index.html`
  - LP本文とセクション構造を管理します。
  - 各セクションには `section id` とコメントを入れています。
  - 写真や数字などの未確定情報はプレースホルダーや `〇〇` 表記にしています。

- `styles.css`
  - 配色、余白、グリッド、カード、プレースホルダー、レスポンシブ表示を管理します。
  - メインカラーは `:root` に変数としてまとめています。
  - セクション単位で切り出しやすいよう、各 `section` は独立した背景と余白を持たせています。

- `script.js`
  - ページ内リンクのスムーススクロールと、ページ上部へ戻るボタンのみ実装しています。
  - CTAの本番動作はSTUDIO側で実装する想定です。

## 編集方法

### テキストを差し替える

`index.html` の各セクション内にある見出し、本文、カード文言を直接編集してください。
セクションごとにコメントを入れているため、該当箇所を探しやすくしています。

### 写真を差し替える

現在は以下のようなプレースホルダーを使用しています。

- `トラック写真予定`
- `道路・都市・トラック写真予定`
- `スタッフ写真予定`
- `本人写真予定`

本番写真に差し替える場合は、該当する `.placeholder` 要素を `img` タグやSTUDIO上の画像配置に置き換えてください。

### 数字を差し替える

`数字で見る、リアルな大塚運送。` セクションと `頑張りとやる気は、収入につながる。` セクションにある `〇〇` を実数に差し替えてください。

### カラーを変更する

`styles.css` 冒頭の `:root` に色をまとめています。

```css
--navy-900: #001b3a;
--navy-800: #002b5c;
--blue-500: #1e88e5;
--blue-300: #66b8ff;
--soft: #f4f8fc;
--text: #061a33;
```

## 確認方法

`index.html` をブラウザで開くと確認できます。
ローカルサーバーを使う場合は、任意の簡易サーバーでこのフォルダを配信してください。
